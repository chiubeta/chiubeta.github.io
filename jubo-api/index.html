Vital Sign

GET /api/vitalLink/vitalsign 列出機構內所有住民一個月內最新一筆生命徵象紀錄

Patient

GET /api/vitalLink/patient 列出機構內所有住民資料
GET /api/vitalLink/patient/{patientId} 取得指定住民資料
POST /api/vitalLink/patient/{patientId} 新建住民資料
PUT /api/vitalLink/patient/{patientId} 更新指定住民資料

User

GET /api/vitalLink/user 列出機構內所有使用者資料
GET /api/vitalLink/user/{userId} 取得指定使用者資料
POST /api/vitalLink/user/{userId} 新建使用者資料
PUT /api/vitalLink/user/{userId} 更新指定使用者資料

Organization

GET /api/vitalLink/organization/branch 取得機構的分區列表
PUT /api/vitalLink/organization/branch 更新機構的分區列表
