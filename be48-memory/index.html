BE48 團歌

Joey

我是真誠、開放的男人

「我不再害怕跌倒，因為每次重新站起來，都是全新的我。」

高階期間，我在三鶯地區推廣兒童程式教育，帶著十幾台筆電往返安親班、社區，為那些沒有接觸過程式設計的孩子，種下一顆希望的種子，讓他們可以在玩樂中學習，看到自己更多的可能性。

同時，有許多家長不僅主動幫孩子報名，甚至會跟孩子一起學習，告訴我說，「要學得比孩子厲害，才不會被問倒！」，我看到的是，他們為了下一代的付出，也讓孩子與家長之間有了更多溫馨互動。

我相信學習不是為了考試與升遷，更不是為了贏在起跑線，而是很單純地，為了看到這個世界有多大，而自己還能走到多遠。
